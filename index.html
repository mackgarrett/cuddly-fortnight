Answer to A9

This CSS file differs from and HTML file because it does not have the header or body features. Also, the CSS file text is more aligned to the left unlike HTML where there is more white space. The CSS is in all black text, while the HTML file has different colored texts. The first familiar things I noticed were the different selectors (h1, h4, etc.) from the reading on CSS, and I also recognized the font-family property and it’s value (arial, helvetica, sans-serif in h1).

Answer to A11

I compared Chrome and Firefox. For the source page, I noticed the html looked the same in both browsers. Also, both browsers displayed the html the same. However, when I zoomed in on the picture of the tree on Chrome, I was able to zoom in much further than I was able to zoom in on the tree in Firefox. Furthermore, when I zoom out as far as possible on both browsers, I noticed that when I resized the zoomed out page, the content on the page was compressed differently than it was when I did the same thing in Chrome.
